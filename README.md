# To-Do Web App

A simple and intuitive **To-Do Web Application** built using **React** (Frontend) and **Node.js** (Backend). The app allows users to create, manage, and delete their tasks efficiently.

## 🚀 Live Demo

[![Live Site](https://img.shields.io/badge/Live%20Demo-Online-green)](https://saksham.todo-web-app.wuaze.com/)

## 🚀 Features
- Add, edit, and delete tasks
- Mark tasks as completed
- Responsive design with modern UI

## 🛠️ Tech Stack
### Frontend:
- **React** (Vite)
- **Tailwind CSS** (for styling)


### Steps to Run:
#### Clone the Repository:
```bash
git clone https://github.com/Saksham-Goel1107/Todo-App.git
```
#### Backend Setup:
```bash
npm install
npm run build
```
